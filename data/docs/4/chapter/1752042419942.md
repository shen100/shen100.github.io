## 外接键盘设置
### 切换系统
| 系统 | 切换 |
| ------- | ------- |
|    Mac OS     |  长按 `Fn + Q`       |
|    Windows     |  长按 `Fn + E`       |
|    Android     |  长按 `Fn + W`       |
|    iOS     |  长按 `Fn + R`       |

### 内置快捷键

### USB 无线的连接方式
关闭键盘电源键，再打开键盘电源键，同时按下 `Fn + 3`  

### 蓝牙的连接方式
关闭键盘电源键，再打开键盘电源键，同时按下 `Fn + 1`
